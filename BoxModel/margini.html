Box sizing

La proprietà box-sizing CSS ci consente di includere il padding e il bordo nella larghezza e altezza totali di un elemento.Per impostazione predefinita, la larghezza e l'altezza di un elemento vengono calcolate in questo modo: larghezza + imbottitura + bordo = larghezza effettiva di un elemento altezza + imbottitura + bordo = altezza effettiva di un elemento Ciò significa: quando si imposta la larghezza/altezza di un elemento, l'elemento spesso appare più grande di quanto è stato impostato (perché il bordo dell'elemento e il riempimento vengono aggiunti alla larghezza/altezza specificata dell'elemento).Dal momento che il risultato dell'utilizzo del box-sizing: border-box; è molto meglio, molti sviluppatori vogliono che tutti gli elementi delle loro pagine funzionino in questo modo. * { box-sizing: border-box; }
